# Лабораторная работа №2: Нейронная сеть на персептронах Розенблатта

## Тема
Проектирование нейронной сети из персептронов для реализации логической схемы.

## Вариант 18
Схема: (x1 AND x2) AND (x1 XOR x3)

## Описание
В работе обучаются отдельные персептроны для базовых логических операций:
- AND (И)
- OR (ИЛИ)
- NAND (И-НЕ)

Затем из них собирается сеть, реализующая XOR и итоговую функцию.

## Структура сети
1-й слой:
├─ AND(x1, x2) → y1
├─ OR(x1, x3) → a
└─ NAND(x1, x3) → b

2-й слой:
└─ AND(a, b) → y2 (это XOR)

3-й слой:
└─ AND(y1, y2) → F

## Файлы
- `logic_gates.py` - обучение базовых элементов
- `perceptron_network.py` - сборка полной схемы

## Запуск

# Сначала обучить базовые элементы (можно пропустить, если веса уже есть)
python logic_gates.py

# Запустить основную сеть
python perceptron_network.py
